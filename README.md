# zeitdb

## Database for time

## Getting Started

### Prerequisites

* A GCC compiler, Make and Python3.

### Compile

``` bash
make
```

### Installation

```bash
make install
```

### Create docs

create virtualenv

```bash
python3 -m venv
pip install -r requirements.txt
source venv/bin/activate
```

build docs in html

```bash
cd docs
make html
```


for more sphinx options use

```bash
make
```
